# To compile example.rohanasantml run:
first cd into examples dir:
```sh
cd ./examples
```

#### If you have Python:

```sh
python3 ../compiler.py ./example.rohanasantml ./generated.html
```

#### If you have V lang:

```sh
v run ../compiler.v ./example.rohanasantml ./generated.html
```
